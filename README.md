# Casos-de-estudios-de-vectores grupo 1
Análisis y diseño de casos de estudios reales utilizando Vectores en JavaScript.
## Introducción

En el desarrollo de los casos de estudio propuestos, se elaboró el código correspondiente empleando los lenguajes HTML y JavaScript, aplicando conceptos fundamentales de programación tales como el uso de vectores, estructuras condicionales, ciclos for y funciones.

El primer caso, denominado "Tienda de Abarrotes", permite al usuario seleccionar los productos deseados, registrar las cantidades adquiridas y generar una factura que incluye el cálculo del subtotal, el IVA, el descuento aplicable y el total a pagar.

El segundo caso, "Cajero Bancario Inteligente", permite registrar depósitos y retiros con el propósito de calcular el saldo disponible.

## Objetivos

- Analizar y diseñar casos de estudio reales mediante la utilización de vectores en JavaScript.
- Elaborar de manera visual una factura en formato HTML.

## Desarrollo

### Caso #1: Tienda de Abarrotes

El usuario selecciona el producto que desea incluir en la lista, visualiza el precio unitario, ingresa la cantidad correspondiente y agrega el producto a la compra.

Al generar la factura, el sistema recorre los vectores mediante un ciclo for con el fin de calcular el subtotal de cada producto y el subtotal general de la compra. Posteriormente, se calcula el IVA correspondiente al 15 %, se verifica mediante una condición si el subtotal supera los veinte dólares para aplicar un descuento del 5 % y, finalmente, se obtiene el total a pagar.

### Caso #2: Cajero Bancario Inteligente

En el caso de los retiros, el sistema verifica que exista saldo suficiente antes de ejecutar la operación. Cada movimiento se almacena en un vector junto con su tipo y monto, lo cual permite mantener un historial de las transacciones realizadas.

Posteriormente, mediante un ciclo for, el sistema recorre el vector de movimientos para mostrar todas las operaciones registradas y calcular un resumen final que incluye el total depositado, el total retirado y el saldo disponible en la cuenta. Para el desarrollo de este sistema se utilizaron variables, vectores, estructuras condicionales if…else, ciclos for y funciones en JavaScript.

## Conclusión

Estos casos permitieron aplicar de manera práctica los conocimientos fundamentales de programación en JavaScript. Se emplearon vectores para almacenar información, ciclos para recorrer los datos y estructuras condicionales para la toma de decisiones conforme a cada situación. Asimismo, se reforzó el manejo de eventos en HTML y la interacción con los elementos de la página mediante el DOM.
